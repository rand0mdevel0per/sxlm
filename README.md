# Quila (QualiaTrace LM)

Dynamic graph reasoning system with 161B effective parameters.

## Status

Phase 0: Foundation (In Progress)

## Requirements

- CUDA 12.0+
- CMake 3.20+
- Python 3.10+
- Protobuf 3.21+

## Build

```bash
mkdir build && cd build
cmake ..
make
```

## Install

```bash
pip install -e .
```
